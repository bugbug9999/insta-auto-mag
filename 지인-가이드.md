# 📖 매거진 만들기 툴 — 사용 가이드 (처음 쓰는 분용)

URL이나 주제를 넣으면 인스타용 매거진 카드(이미지 여러 장)를 자동으로 만들어주는 툴이에요.
**당신의 클로드(Claude) 구독으로 돌아갑니다 — 따로 결제할 거 없어요.**

준비물: 맥(Mac) + 클로드 구독. 끝.

---

## 1단계. 클로드 코드(Claude Code) 준비 — 한 번만
이 툴은 "클로드 코드"라는 걸 통해 켜요. 이미 쓰고 있으면 2단계로.

처음이라면 → **클로드 앱(또는 claude.ai)에 이렇게 물어보세요:**
> 맥에 Claude Code(클로드 코드) 설치하는 법 알려줘. 한 단계씩 천천히.

시키는 대로 따라 하면 돼요. (당신 구독에 포함돼 있어서 추가 비용 없음)

---

## 2단계. 받기 + 켜기 — 클로드한테 통째로 시키기 (제일 쉬움)
다운로드도 손으로 할 필요 없어요. **클로드 코드**를 열고 아래를 **그대로 복사해 붙여넣기:**

```
github.com/bugbug9999/insta-auto-mag 를 ~/Desktop 에 git clone 해줘 (이미 있으면 git pull).
그 폴더(insta-auto-mag)로 가서:
1) python3 -c "import PIL" 로 Pillow 확인, 없으면 pip3 install Pillow (실패하면 pip3 install --break-system-packages Pillow)
2) node server.js 를 백그라운드로 실행
3) 서버 로그에 listening 뜨면 http://localhost:3080 을 브라우저로 열어줘
다 되면 "준비 완료"라고 말해줘. 에러 나면 원인 고쳐서 다시 시도해줘.
```

클로드가 **받아오고 → 깔고 → 실행 → 브라우저 열기**까지 다 합니다. (레포 공개라 인증 불필요)

### 다음부터 다시 켤 때
```
~/Desktop/insta-auto-mag 에서 node server.js 를 백그라운드로 켜고 http://localhost:3080 을 열어줘
```

---

## 4단계. 사용하기
브라우저에 뜬 화면에서:
1. **칸에 기사 URL을 붙여넣기** (또는 만들고 싶은 주제)
2. 그 아래 **"편집 방향" 칸**에 원하는 느낌 적기 (선택) — 예: `더 쉽게 풀어서`, `해외 진출 스토리로`
3. **생성** 버튼 클릭 → 잠시 기다리면 매거진 카드가 만들어져요
4. 카드 편집하고 내보내기(Export)

> 💡 "API 키" 입력칸은 **무시해도 돼요.** 당신 클로드 구독으로 돌아가니까요.
> (만약 "claude를 못 찾는다"는 에러가 나면 → 1단계 클로드 코드 설치가 안 된 거예요.)

---

## 끄기 / 다시 켜기
- **끄기**: 검은 창(터미널)을 닫으면 꺼져요.
- **다시 켜기**: 3단계 다시 (더블클릭 또는 프롬프트).

## 안 될 때 — 클로드한테 그대로 붙여넣기
```
매거진 툴 서버가 안 켜져. 이 폴더에서 node server.js 를 실행하고
에러 로그를 보고 원인을 고쳐서 http://localhost:3080 이 열리게 해줘.
```
