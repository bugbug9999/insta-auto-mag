# 감성 매거진 구조 DNA 분석 (실포스트 18개·본문 슬라이드 80장 검안)

> 수집: 레퍼런스 계정 18포스트(커버+캡션+본문 슬라이드 80여 장) 분석 기반.
> 비즈니스/테크 축 + 인물/컬처 축. 공식은 동일.

## 1. 썸네일(커버) 구조
- **인물 이슈 = 얼굴 클로즈업 100%** (화면 30~60%). 흑백 or 원색. 와이드샷/뒷모습 0장.
- 인물 없으면: 로고 충돌(구글G+메타∞), 상징 오브젝트(로켓), 컨셉 일러스트(시시포스+선물상자).
- 합성 연출 적극적: 우주복에 이건희 얼굴 합성, 알트먼+트럼프+샌더스 콜라주+형광 그린 배경.
- 레이아웃: 사진 풀블리드 + 하단 다크 그라데이션 + **좌하단 흰 Heavy 고딕 2줄** + 그 위 작은 키커(포인트색).

## 2. 헤드라인 문법 (8공식 — 전부 '사람 말', 신문 제목 0건)
기존 6공식(결단인용/의외고백/의인화+밈/독자질문/직설명령/충격사실+?)에 실측 추가:
- **감탄 깨달음형**: "거제 야호, 될 수 밖에 없었구나" (분석 포스트의 결론 선출시)
- **1인칭 선언형**: "나는 원고지 앞의 만화 노동자다" (인물 서사)

## 3. 본문 템플릿 4종 (콘텐츠 포맷이 템플릿을 결정한다)
| 포맷 | 템플릿 | 배경 | 사진/그림 | 강조 |
|---|---|---|---|---|
| 뉴스/브리프 | **다크 뉴스형** | 다크네이비 | 인셋 보도사진 카드 | 포인트색 소제목+연두 하이라이트 |
| 어록/원칙 | **라이트 아카이브형** | 아이보리(#F4F1EA대) | 상단 아카이브 사진 카드(~50%h, 컬러/흑백) | **검정 볼드만**(형광 0) |
| 인물서사/탐정분석 | **라이트 스토리형** | 아이보리 | 기울인 폴라로이드 콜라주 2~3장, 캡처 증거 | 연두 하이라이트(라이트판)+볼드 |
| 배움/프레임 | **다크 컨셉형** | 검정 | 오렌지 모노톤 은유 일러스트(풀블리드와 교차) | 볼드+밑줄, 표/다이어그램 허용 |

## 4. 내용 전개 (슬라이드 단위 실측 패턴)
- **어록형(이건희 15장)**: 커버(의외 고백) → 슬라이드당 어록 1개 + 해설 3~6문장 → 마지막 문장 항상 **볼드 펀치라인**("늦으면 못 따", "다시, 바꿔.")
- **넘버링 리스티클(이병철 10장)**: 커버(직설 명령) → "1.~8. 명령형 소제목(볼드)" + 해설 3줄 + 볼드 마무리. 저장 유도형.
- **탐정/역주행형(거제야호 9장)**: ①결과 먼저(대박) ②"그런데 기적이라기엔" 반전 의심 ③거슬러 올라가기(누적 숫자: 2년 1,500개 라이브) ④디테일 한 방(7시간 라이브) ⑤재정의 엔딩(운이 아니라 축적). **결론을 먼저 보여주고 원인을 추적하는 구조.**
- **인물서사형(허영만 10장)**: 1인칭 선언 커버 → 연대기 장면들 → 습관/원칙 → 결과 → "지금도 ~한다" 현재형 마무리.
- 공통: 슬라이드당 메시지 1개. 다음 장을 넘기게 하는 끝맺음(질문/미완 문장) 빈번.

## 5. 문장 구조·리듬 (실측)
- 초단문 다수(5~15자). **같은 동사 3연속 반복 리듬**: "바꿔. 철저히 바꿔. 끝까지 바꿔."
- 어록 인용=반말 그대로, 해설=~입니다/~이야 혼용(포스트 톤 단위로 통일).
- 숫자는 항상 구체("2년에 1,500개", "7시간"). 큰 수는 비교로 체감("국가 단위").
- 인용구는 따옴표 박스로 분리. 마지막 줄 펀치라인은 거의 항상 볼드.

## 6. 타이포그래피
- 전 슬라이드 고딕 단일 패밀리(Apple SD Gothic Neo급). 커버 헤드 Heavy 80~90px급, 본문 소제목 Bold ~54px, 본문 38~42px, 줄간 1.5~1.7.
- 본문 가운데 정렬 일관(라이트 스토리형 일부 좌정렬 혼용).
- 강조 위계: 다크=연두 형광+포인트색 소제목 / 라이트=검정 볼드(원칙: 형광 금지) / 공통=밑줄은 드물게.
- 워터마크: 우하단 로고 박스(레퍼런스/레퍼런스), 커버는 하단 중앙.

## 7. 우리 시스템 반영 (이 분석의 적용처)
- style/bible.md v3: 8공식+포맷별 전개+문장 리듬.
- make_card.py: **라이트 아카이브 템플릿 추가**(theme=light) — 어록/인물 포스트가 콘텐츠 기둥인데 템플릿이 없었음.
- outline 단계에서 postFormat(news|quote|listicle|detective|profile|learning) 선택 → 템플릿/리듬 자동 매핑.

# v4 — 레퍼런스 현행 피드 실측 (2026-06-13, 최신 4포스트 36장 수집·8장 정밀 검안)

> 수집: 레퍼런스 최신 4포스트(마크 앤드리슨 9장 / AI·VC 해설 8장 / 플라야 멤버십클럽 9장 / 사이먼 킴 10장), ig-fetch.
> **v3 분석(레퍼런스 합산) 대비 레퍼런스 단독 피드는 이미 다른 단계로 진화해 있음** — 다크네이비+형광 스타일이 아니라 밝은 배경+검정 타이포 중심.

## 1. 템플릿 4계열 (포스트 단위로 1계열 고정)
- **해설형(상사진+하카드)**: 상단 풀블리드 사진 ~60% + 하단 라이트그레이(#E8E8E6급) 텍스트 카드 전폭. 우리 뉴스형과 같은 골격, 색만 밝음.
- **스토리형(아이보리)**: 아이보리 전면 + 중앙 사진(러프 테두리) + 상하 텍스트 가운데 정렬. 포인트 컬러 **빨강 1문장**.
- **인물 서사형(블랙)**: 검정 전면 + 흰 타이포 블록 + 하단 대형 인물 사진. 1인칭 반말.
- **리스트형(화이트)**: 흰 전면 + 상단 텍스트 블록 + 하단 사진. 번호 소제목.

## 2. 타이틀(소제목) — 유저 지적의 원본 근거
- **본문 슬라이드 타이틀은 전 장 예외 없이 한 줄.** "1. 이제 누구나 만들 수 있는 시대입니다"(15자+)도 폭을 줄여서라도 한 줄.
- **번호 prefix "1. 2. 3."** 빈번(해설형·리스트형). 검정(흰) 볼드, 줄바꿈 없음.
- 커버 헤드라인만 2줄 허용(의미 단위 절단, "프라이빗 멤버쉽 클럽, / 한국에도 있습니다").

## 3. 본문 문장 — 띄어쓰기/줄바꿈
- **인위적 초단문 줄바꿈 없음. 완결 문장이 카드 폭에 맞춰 자연 wrap.** 한 문단 2~4줄, 문단 2~3개, 문단 사이 빈 줄 반 칸.
- 짧은 한 줄 문단을 리듬 변화로 끼움("이제 혼자 다 만들고 돈까지 법니다.").
- 줄 폭을 넓게 쓴다 — 좌우 여백 최소, 카드 전폭 사용.

## 4. 말투 2계열
- **해설형 = 정중 구어체**: ~입니다/~같습니다/~니까요/~고요. 설명 흐름 + 마지막 단정 펀치.
- **인물형 = 반말 단정체**: "~다." ("서울에서 태어나 미국에서 자랐다. 두 나라가 내 안에 다 있었다.")
- 공통: **마지막 문단 = 볼드+밑줄 펀치라인 1문장** ("결과를 만든 사람을 채용해야 한다.").

## 5. 강조 체계
- **형광 하이라이트 없음.** 강조 = 검정(흰) **볼드+밑줄**. 아이보리 템플릿만 빨강 문장 1개 추가.
- 워터마크: 우하단 소형 검정 박스 "레퍼런스"(흰 글자). 커버는 하단 중앙.

## 6. 우리 시스템과의 갭 (적용 우선순위)
1. (적용) 소제목 한 줄 강제 + 번호 prefix 허용 — 렌더러 폭맞춤 축소 분리 완료(2026-06-13).
2. (적용) 본문 22자 초단문 끊기 폐지 → 완결 문장 + 렌더러 자동 wrap.
3. (적용) 마지막 문단 볼드+밑줄 펀치 규칙.
4. (유저 결정 필요) 배경 다크네이비 유지 여부 — 레퍼런스 현행은 그레이/아이보리/화이트/블랙이고 형광 라임 강조가 없음. 유저가 이전에 네이비 통일을 지시했으므로 색 전환은 보류.
